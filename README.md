# Personal Portfolio Website & Brand Identity

A handcrafted personal portfolio with a distinctive Art Deco 
visual identity — designed and built from scratch.

## Overview
Full portfolio website featuring a custom brand system built around 
Art Deco principles: geometric structure, black and gold palette, 
serif typography, and high-contrast layouts.

## Features
- Art Deco aesthetic — black, gold, and serif typography throughout
- Matching business card design with consistent brand language
- Brand guidelines covering typography, colour palette, and layout rules
- Responsive layout across desktop and mobile
- Sections: About, Skills, Projects, Certifications, Contact

## Design System
| Token | Value |
|-------|-------|
| Primary | #0D1B2A (Deep Navy) |
| Accent | #C9A84C (Warm Gold) |
| Background | #F7F8FA |
| Display Font | Serif (Art Deco style) |
| Body Font | Helvetica / Sans-serif |

## Tech Stack
- HTML5, CSS3
- Canva (business card and brand assets)
- Custom typography and layout — no CSS frameworks

## Status
Live and actively maintained.
